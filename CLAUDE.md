# Свої (Svoi) — CLAUDE.md

## Про проєкт
Утилітарна платформа для української діаспори за кордоном. НЕ соціальна мережа.
Допомагає знайти своїх: оголошення, івенти, опитування, карта, контакти.

## Техстек
- Flutter 3.44+ / Dart 3.12+
- State management: Riverpod 3.x (Notifier + NotifierProvider)
- Firebase: Auth, Firestore, Cloud Messaging, Hosting (ще не підключено)
- Карта: google_maps_flutter (ще не підключено)
- Шрифт: Inter (google_fonts)
- Package ID: app.svoi

## Структура
- lib/core/ — теми, роутер, спільна логіка
- lib/features/ — фічі по папках (home, map, saved, profile)
- lib/shared/ — спільні віджети і утиліти

## Дизайн
- Стиль: мінімалізм (Monobank/Дія)
- Основний колір: #6C4DE6 (індиго-фіолетовий)
- Три теми: світла (за замовчуванням), темна, системна
- Етнічні елементи тільки в splash/іконка/онбординг/empty states

## Навігація
- Bottom nav: Головна, Карта, Збережене, Профіль
- Головна: 3 верхніх таби — Оголошення, Івенти, Опитування

## Контент
- 10 категорій оголошень: Послуги, Житло, Робота, Діти, Речі, Документи, Транспорт, Тварини, Краса, Юридичне
- 10 категорій івентів: Зустрічі, Спорт, Діти, Свята, Навчання, Допомога, Культура, Розваги, Подорожі, Бізнес
- Опитування з гео-скоупом (місто/країна/всі)

## Правила роботи
- Мова інтерфейсу: тільки українська
- Коментарі в коді: українською в ключових місцях
- НЕ починай писати код без явного дозволу від розробника ("починай")
- Після змін завжди запускай flutter analyze
- Кожну логічну зміну комітимо окремо

## Запуск (Польща першою, Android першим)
- Цільова аудиторія: українська діаспора, запуск в Польщі
- Android перший, iOS пізніше
